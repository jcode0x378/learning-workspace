# Format

- readability 可讀性

## 留白 spacing

- 空格
- 換行

```js
// object
const foo = { a: 1 }; // inline object need inside space between
const bar = [1, 2]; // array doesn't need inside space between
console.log(1, 2); // parenthesis don't need too

// keyword space
if () {}

// statement (purple) line break
switch () {}

// operator
1 + 1;
foo = 1;

let foo;
const bar = 1;

function foo() {} // function line break okay

// keyword: purple (non-declaration), pink (declaration)

```
