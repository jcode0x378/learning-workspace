# Term

- program 程式
- programming 程式設計
- application 應用程式 APP

## expression 表示式／表達式

- 最後會 evaluate 出一個值（ value ）
- 可以放在 RHS （ right hand side ）

## statement 陳述式 （語句）

- 相對複雜的（紫色）
- 不會 evaluate 出一個值（其實他會）
- 不可以放在 RHS
